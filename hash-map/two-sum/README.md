# Two Sum

## Условие

Дан массив чисел `nums` и число `target`. Найди два индекса `i` и `j` такие, что:

```
nums[i] + nums[j] == target
```

## Ключевая идея

Для каждого элемента `x` нам нужна его пара `target - x`. Вместо перебора всех пар за O(n²) — храним уже встреченные числа в хеш-таблице и проверяем наличие пары за O(1).

## Сложность

|                  | Время  | Память |
|------------------|--------|--------|
| Двойной цикл     | O(n²)  | O(1)   |
| Хеш-таблица      | O(n)   | O(n)   |
