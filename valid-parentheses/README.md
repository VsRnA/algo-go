# Valid Parentheses

## Условие

Дана строка из символов `()[]{}`. Верни `true`, если последовательность корректна — каждая открывающая скобка закрыта соответствующей закрывающей в правильном порядке.

## Ключевая идея

Идём слева направо. Открывающую скобку кладём в стек. Встретили закрывающую — смотрим на верхушку стека: если там соответствующая открывающая, снимаем её. Если нет — строка невалидна.

В Go стек реализуется через слайс:

```go
stack = append(stack, c)
top  := stack[len(stack)-1]
stack = stack[:len(stack)-1]
```

## Сложность

| | Время | Память |
|---|---|---|
| Решение со стеком | O(n) | O(n) |

В худшем случае все символы — открывающие скобки, стек вырастает до размера строки.

## Граничные случаи

- Пустая строка `""` → `true` (стек пуст, условие выполнено)
- Одна скобка `"("` → `false` (стек не пуст в конце)
- Закрывающая при пустом стеке `")"` → `false` (проверяем `len(stack) == 0` перед peek)
- `"([)]"` → `false` (скобки есть, но порядок закрытия нарушен)
